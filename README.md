# SWP391 Team 7 — MediCare AI Clinic

Dự án phòng khám thông minh tích hợp AI (SWP391).

## Cấu trúc

```
SWP391_Team7/
└── Medicare/
    ├── BE/          # Node.js + Express + MongoDB
    ├── FE/          # React + TypeScript + Tailwind
    ├── package.json # Chạy cả BE + FE
    └── README.md    # Hướng dẫn chi tiết
```

## Chạy nhanh

```powershell
cd Medicare
npm run install:all
copy BE\.env.example BE\.env
copy FE\.env.example FE\.env
npm run dev
```

- Frontend: http://localhost:3000  
- Backend: http://localhost:8000  

Xem thêm trong [Medicare/README.md](./Medicare/README.md).
